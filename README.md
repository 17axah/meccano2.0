# Meccano Builder 2.0 :v:

## Установить зависимости
```bash
$ npm install
```

## Запуск в режиме dev
```bash
$ npm run dev
```

## Сборка проекта в режиме prod
```bash
$ npm run prod
```

## Запуск собранного проекта в режиме prod
```bash
$ npm run serve
```

## Создание нового компонента
```bash
$ npm run add
```
